前端学习笔记

Posted on 2022-07-29 In Learning , front-end Disqus:
Symbols count in article: 1.7k Reading time ≈ 2 mins.

前端主要学习html,CSS,JavaScript

[TOC]

HTML

HTML(Hypertext Makeup Language)，即超文本标记语言，主要负责网页内容的展示，大小写不敏感。

标记

一般以<标签名>…</标签名>的形式成对出现，自结束标签无需结束符

常见的标签

  • 文档声明[html5]：放在顶端

  • 网页根标签：<html>...</html>

  • <head>...</head>非用户可见，一般为告诉浏览器属性

  • head中的标题/浏览器头部标题：<title>text</title>

    • <body>...</body>用户可见，具体内容

    • 标题：<hn>text</hn>,n=1~6

    • 段落：<p>text</p>

    • 超链接：<a>text</a>

    • 换行：<br>

    • 粗体：<b>text</b>

      Atten:<strong>text</strong>也有加粗的效果，但是还表示重要呈现

    • 按钮：<button>text</button>

    • 文档中的节：<div>text</div>

    • 文本中的节：<span>text</span>

    • 斜体：<i>text</i>

      Atten:<em>text</em>也有斜体效果，同样表示重要呈现

    • 输入：<input>

    • 高亮：<mark>text</mark>

    • 下标：1<sub>text</sub>

    • 上标：1<sup>text</sup>

    • 删除线：<del>text</del>

    • 插入/下划线：<ins>text</ins>

标签属性

在开始标签中设置，属性是键值对结构(“key”=”value”)

常见属性

  • id(唯一)

  • style:指定行内样式，多个样式以;分隔

    eg.:style=”color:red;text-align:center”

  • title：额外信息/工具条

  1. 注释

CSS

CSS(Cascading Style Sheets)，主要负责网页布局，描述HTML/XML的呈现，即其如何被渲染。

eg.:

p{

​ font-size: 20px;

​ color:red;

}

<p>text</p>

p:选择器

font-size/color:属性，冒号后为其值

大括号括着的为声明，声明以;结束

选择器

  • id选择器

    以#定义，不以数字开头(避免在Mozilla/Firefox不起作用)

    eg.: #test{

    ​ text-align:center;

    ​ color:red;

    }

    <p id=”test”>text</p>

  • class选择器

    以.显示，多个用空格区分，类名第一个字符非数字

    eg.:

    .center{

    ​ text-align:center;

    }

    <p class=”center”>text</p>

注释：/*text*/

插入样式表

  • 外部样式表(external style sheet)

    一般用于同一样式需用于很多页面，通过mystyle.css文件改变站点整体样式【页面使用<link>标签链接到样式表，该标签位于文档头部<head>中】

    <link rel="stylesheet" href="mystyle.css">

  • 内部样式表(internal style sheet)

    一般用于单个文件，利用<style>标签在头部文件内部定义。

  • 内联样式(inline style)

Atten:多重样式优先级
内联>内部>外部>默认
