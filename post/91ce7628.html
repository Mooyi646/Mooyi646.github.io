HTML

HTML(Hypertext Makeup Language)，即超文本标记语言，主要负责网页内容的展示，大小写不敏感。

标记

一般以<标签名>…</标签名>的形式成对出现，自结束标签无需结束符

常见的标签

  • 文档声明[html5]：放在顶端

  • 网页根标签：<html>...</html>

  • <head>...</head>非用户可见，一般为告诉浏览器属性

    • head中的标题/浏览器头部标题：<title>text</title>

    • HTML文档元数据：<meta>,可说明编码类型，描述(discription)，作者(author)等等

      <meta charset="utf-8">

      <meta name="author" content="moo1">

    • 链接：<link>，定义文档与外部资源的关系，空元素，只包含属性，常用于来链接样式表。

      • 属性：

        1. herf：被链接文档的位置
        2. rel：当前文档与被链接文档之间的关系，必须
        3. type：type 属性规定被链接文档/资源的 MIME 类型。只有当设置了 href 属性时，才能使用该属性。<link> 标签常用的 MIME 类型是 “text/css”，它规定样式表。

        更多与link属性相关可点击查看。

  • <body>...</body>用户可见，具体内容

  • 标题：<hn>text</hn>,n=1~6

  • 段落：<p>text</p>

  • 超链接：<a>text</a>，一般与herf属性一起使用

    welcome to mooyi’s channel!

  • 换行：<br>

  • 粗体：<b>text</b>

    Atten:<strong>text</strong>也有加粗的效果，但是还表示重要呈现

  • 按钮：<button>text</button>

  • 文档中的节：<div>text</div>

  • 文本中的节：<span>text</span>

  • 斜体：<i>text</i>

    Atten:<em>text</em>也有斜体效果，同样表示重要呈现

  • 输入：<input>

  • 高亮：<mark>text</mark>

    text

  • 下标：1<sub>text</sub>

    1text

  • 上标：1<sup>text</sup>

    1text

  • 删除线：<del>text</del>

    text

  • 插入/下划线：<ins>text</ins>

    text

更多标签可点击查看。

标签属性

在开始标签中设置，属性是键值对结构(“key”=”value”)

常见属性

  • id(唯一)

  • style:指定行内样式，多个样式以;分隔

    style="color:red;text-align:center"

    style-showing.

  • title：额外信息/工具条

    <p><ins title="格外信息">move</ins> on this</p>

    move on this

更多属性可点击查看。

注释

<!--this is unshown.-->

HTML事件

事件可以触发浏览器行为，与JS联合使用。

表示：event="script()"

窗口事件属性(Window Event Attributes)

窗口触发，适用于<body>标签

  • onresize：调整窗口大小时运行脚本

    <body onresize="Function()">

  • 文档触发，窗口失焦…

表单事件(Form Events)

表单中触发，HTML元素需在form表单内。

键盘事件(Mouse Events)

  • onkeydown：按下键盘
  • onkeypress：按下并松开
  • onkeyup：松开按键

鼠标事件(Mouse Events)

  • onclick：单击
  • ondblclick：双击
  • ondrag：拖动元素

…

通过视频（videos），图像（images）或者音频（audio） 触发该事件，多应用于 HTML 媒体元素比如 <audio>, <embed>, <img>, <object>, 和<video>:

其他事件

更多事件可点击查看。

CSS

CSS(Cascading Style Sheets)，主要负责网页布局，描述HTML/XML的呈现，即其如何被渲染。

<p>text</p>

p:选择器

font-size/color:属性，冒号后为其值

大括号括着的为声明，声明以;结束

选择器

id选择器

以#定义，不以数字开头(避免在Mozilla/Firefox不起作用)

eg.: #test{

​ text-align:center;

​ color:red;

}

<p id=”test”>text</p>

class选择器

以.显示，多个用空格区分，类名第一个字符非数字

eg.:

.center{

​ text-align:center;

}

<p class=”center”>text</p>

注释：/*text*/

插入样式表

外部样式表(external style sheet)

一般用于同一样式需用于很多页面，通过mystyle.css文件改变站点整体样式【页面使用<link>标签链接到样式表，该标签位于文档头部<head>中】

<link rel="stylesheet" href="mystyle.css">

内部样式表(internal style sheet)

一般用于单个文件，利用<style>标签在头部文件内部定义。

内联样式(inline style)

<p style="color:sienna;margin-left:20px">text</p>

text

Atten:多重样式优先级
内联>内部>外部>默认

背景

  • background-color
  • background-image
  • background-repeat：水平或垂直方向平铺或不平铺(repeat-x/repeat-y/no-repeat)
  • background-attachment：是否固定/滚动
  • background-position：起始位置

背景可简写属性，属性顺序如上顺序，属性可不全部使用。

body {background:#ffffff url('img_tree.png') no-repeat right top;}

对齐

元素居中对齐

使用margin: auto

图片居中对其需放入块元素中

文本居中对齐

text-align: center

左右对齐-定位

position:absolute

注释：绝对定位元素会被从正常流中删除，并且能够交叠元素。

Atten: 当使用 position 来对齐元素时, 通常<body>元素会设置 margin 和 padding 。 这样可以避免在不同的浏览器中出现可见的差异。

左右对齐-float

float:left/right

Atten：如果子元素的高度大于父元素，且子元素设置了浮动，那么子元素将溢出，这时候你可以使用 “clearfix(清除浮动)” 来解决该问题。可以在父元素上加overflow:auto来解决子元素溢出。

垂直居中-padding

垂直居中-line-height

垂直居中-position&transform

JavaScript

JavaScript是一种脚本语言，插入HTML页面后，由浏览器执行。

  • 直接写入HTML输出流：document.write(“<p>text</p>“);

    Atten:document.write()只能在HTML输出流中使用，如在文档加载后使用，会覆盖整个文档。
